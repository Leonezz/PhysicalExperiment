# 冉绍尔--汤森实验

## 一、	实验目的

1. 了解电子碰撞管的设计原则，掌握电子与原子的碰撞规则和测量的原子散射截面的方法。
2. 测量低能电子与气体原子碰撞的散射几率$P_s$与电子速度的关系。
3. 测量气体原子的有效弹性散射截面$Q$与电子速度的关系，测定散射截面最小时的电子能量。
4. 验证冉绍尔-汤森效应，并学习用量子力学理论加以解释。
